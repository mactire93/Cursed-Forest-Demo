------- PL -------

Cursed Forest - Demo

Gra powstała w celach naukowych, jeden z moich pierwszych projektów podczas nauki PyGame.

Wszelkie zasoby gry posiadają otwartą licencję, dlatego dodałem odnośniki w kodzie gry.

Gra nie służy celom komercyjnym.

Projekt nie został skończony, choć nie wykluczam ponownego podejścia do projektu. W chwili obecnej skupiałem się na nauce innych dziedzin Python, niemniej PyGame i tworzenie gier jest szalenie przyjemne. 

Kontrola:

Spacja - skok
A,D - ruch lewo, prawo
S - strzał (rzut kamieniem)

Uważaj na zombie, wydają się być leniwe ale w pewnym momencie przyspieszają!

------- ENG -------

Cursed Forest - Demo

The game was created for research purposes, one of my first projects while learning PyGame.

All game assets are openly licensed, so I added links in the game code.

The game is not for commercial purposes.

The project is not finished, although I do not rule out approaching it again. At the moment I was focused on learning other areas of Python, nevertheless PyGame and game development is wildly enjoyable. 

Controls:

Spacebar - jump
A,D - move left, right
S - shot (stone throw)

Watch out for zombies, they seem to be lazy but at some point they speed up!
